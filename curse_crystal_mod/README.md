## 🔮 СПОСОБНОСТЬ ПКМ — Луч Апокалипсиса

При нажатии **ПКМ** с мечом в руке:

### 1️⃣ Крылья из фиолетового огня
- Из спины игрока вырываются крылья из фиолетовых частиц
- Верхние крылья смотрят **вверх**, нижние — **вниз**
- 6 "перьев" с каждой стороны, дуги из DustParticle + Flame

### 2️⃣ Луч Апокалипсиса
- Из меча вырывается **фиолетовый луч** в направлении взгляда
- Луч состоит из 3 слоёв частиц (ядро + ореол + внешний ореол)
- **Дальность:** 128 блоков
- **Урон:** 99,999,999 ❤ — мгновенная смерть всего живого
- **Ломает все блоки** кроме бедрока
- В конце луча — грандиозный взрыв частиц

### 3️⃣ Меч горит фиолетовым
- Во время выстрела игрок загорается (визуально)
- Fire Resistance защищает от урона — только визуал!
- Dragon Breath эффект вдоль всего луча

**Кулдаун:** 10 секунд

## 🔧 Установка

1. [Fabric Loader 0.14+](https://fabricmc.net/)
2. [Fabric API 0.91+](https://modrinth.com/mod/fabric-api)
3. Скинь `.jar` в папку `mods/`

## 🏗️ Сборка

```bash
./gradlew build
# → build/libs/curse-crystal-mod-1.0.0.jar
```
